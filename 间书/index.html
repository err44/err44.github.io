    简介

    获取页面HTML，订阅元素转为RSS，搭配本地web实现订阅动态。

    #Python #RSS

    功能

    版本
    0.5+ 动态页面
    0.1+ HTML元素转RSS
    静态页面
    单代理
    预定实现
    多代理
    GUI
    UI隐藏

    说明

    rss/

    无后缀xml保存目录

    data/

    model.ini 订阅模板，可动态修改

    版本 变量名 值 说明
    0.5+ 动态页面 0
    1    		 否
    是
    0.1+ 过滤选项 0
    1    		 标题去重
    链接去重
    保存目录 “”
    “A/“
    “A”    		 自动获取
    127.0.0.1/rss/A/666
    127.0.0.1/rss/A
    headers 一般留空
    ID ‘/‘
    留空
    “123/video”    		 单页面
    跳过模板
    适用于识别ID在链接中间的网站

    config.ini 配置文件，修改需要重启软件

    save.ini 重复过滤

    以上三个 ini 本质都为Python dict

    运行组件

    安装 Python 3.8.6

    https://www.python.org/downloads/release/python-386/

    （务必add Python x to PATH）
